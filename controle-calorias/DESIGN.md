# Controle de Calorias - Design System

## Direção visual

Aplicativo moderno, limpo e mobile-first, com dark mode como padrão.

## Paleta inicial

- Fundo principal: quase preto
- Fundo dos cards: cinza escuro
- Cor principal: verde energia
- Cor secundária: azul suave
- Texto principal: branco suave
- Texto secundário: cinza claro
- Erro: vermelho
- Sucesso: verde
- Aviso: amarelo

## Componentes base

- Botão primário
- Botão secundário
- Botão de perigo
- Cards
- Inputs
- Barra de progresso
- Badges
- Loading
- Toast

## Espaçamentos

Escala padrão:

- 4px
- 8px
- 12px
- 16px
- 24px
- 32px
- 48px

## Bordas

- Pequena: 8px
- Média: 12px
- Grande: 20px

## Sombras

Cards devem ter sombra suave, sem exagero.

## Mobile-first

Todas as telas devem funcionar bem primeiro no celular.

Depois ajustamos para desktop.
