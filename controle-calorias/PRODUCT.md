# Controle de Calorias

## Objetivo

Criar um aplicativo mobile para controle alimentar utilizando Inteligência Artificial para reconhecer refeições através de fotos, estimar calorias e acompanhar a evolução nutricional do usuário.

## Público-alvo

Inicialmente uso pessoal.

Futuramente qualquer usuário interessado em acompanhar alimentação, emagrecimento, ganho de massa ou manutenção de peso.

## Objetivos da versão 1

- Login Google
- Dashboard diário
- Cadastro de refeições
- Upload de foto
- Análise por IA
- Histórico
- Perfil do usuário

## Diferenciais

- IA para reconhecimento de alimentos
- Interface simples
- Mobile First
- PWA
- Dados privados
